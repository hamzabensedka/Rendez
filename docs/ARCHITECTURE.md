# Planity Clone — Architecture

## Overview
A salon and beauty booking marketplace MVP with Apple-style UI/UX. Clients can browse salons without login, search by location or map, view business details, book appointments with service and time-slot selection, and manage upcoming bookings and favorites. Built as an Nx monorepo with an Expo React Native mobile app and a NestJS + Prisma + PostgreSQL API. Admin dashboard, payments, and background job processing are planned but not yet implemented.

## Tech Stack
- **Frontend:** Expo, React Native, TypeScript, Expo Router, TanStack React Query, React Native Reanimated
- **Backend:** NestJS, Prisma, PostgreSQL, PostGIS, Redis
- **DevOps:** Nx, pnpm, Docker Compose, GitHub Actions, Supabase
