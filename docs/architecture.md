Mobile App (`apps/mobile`)\n- **Presentation Layer**: Screens built with React Native components; navigation via Expo Router (file‚Äëbased routing).\n- **State Management**: TanStack React Query for server‚Äëstate (fetching, caching, optimistic updates).\n- **Animations & Gestures**: React Native Reanimated 2 for smooth transitions (e.g., map drag, modal presentations).\n- **API Communication**: Thin service layer (`services/apiClient`) that wraps `axios`/`fetch` and integrates with React Query hooks.\n- **Authentication**: JWT stored securely (expo-secure-store); refresh token flow handled via a custom `useAuth` hook.\n- **Offline First**: Query results cached; background sync via React Query‚Äôs refetch intervals.\n- **Design System**: Consumes `libs/ui` for consistent theming, accessibility, and responsive layout.\n\n### 2.
