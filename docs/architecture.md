This document defines a clean, scalable architecture using a pnpm + Nx monorepo, an Expo React Native mobile client, and a NestJS backend with PostgreSQL/PostGIS, Redis, and Supabase. The design enforces separation of concerns via domain modules, shared libraries, and explicit service boundaries.\n\n## 2. Technology Stack\n- Mobile: Expo, React Native, TypeScript, Expo Router, TanStack React Query, React Native Reanimated.\n- Backend: NestJS, Prisma, PostgreSQL + PostGIS, Redis (cache + BullMQ), Supabase (Auth, Storage, Push).\n- Tooling: Nx, pnpm, Docker Compose, GitHub Actions, EAS Build, Jest.\n\n## 3. Monorepo Layout (Nx + pnpm)\nWe use a single Nx workspace managed by pnpm.
