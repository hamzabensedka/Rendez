Technology Stack\n- **Mobile**: Expo, React Native, TypeScript, Expo Router, TanStack React Query, React Native Reanimated\n- **Backend**: NestJS, Prisma ORM, PostgreSQL with PostGIS extension, Redis, Service\n   - Admin Service\n   - Provider Portal Service\n   - Payment Service (gateway adapter)\n   - Map Service (geospatial utilities)\n4. **Data Layer** – Prisma models backed by PostgreSQL + Services → Repositories (Prisma).\n- DTOs validated via class‑validator & class‑transformer.\n- Global exception filter for consistent error responses.\n- JWT AuthGuard protecting routes; role‑based guards for Admin/Provider.\n- Pagination, filtering, sorting via query parameters.\n- Webhook endpoints for payment gateway and external services.\n\n## Mobile Architecture\n- **Navigation**: Expo **Geospatial**: PostGIS GIST indexes on `location` for fast nearest‑neighbor queries.\n- **Rate Limiting & Security**: NestJS ThrottlerGuard, helmet, CORS, input sanitization.\n\n## Summary\nThis
